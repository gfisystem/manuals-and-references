Each algorithm has up to 8 unique parameters that are adjusted using the PARAM knob

Algorithms are selected using this knob.

These two knobs control 3 parameters that are common to all algorithms:

· TIME :  Delay Time (for delay) or Predelay (for reverb).

· WET  :  effects signal level.

· DRY   :   clean, unprocessed signal level.
