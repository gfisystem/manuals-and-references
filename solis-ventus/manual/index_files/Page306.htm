5.2  Aux Switches

External switches are absolutely wonderful and effective for unlocking the rich potential of user’s control and access to the various features and functionalities. Aux switch configuration is done in the Global Settings menu (see page 24). Here are some of the functions you can assign to external switches:

 

5.1  MIDI

You can use any standard MIDI controller capable of sending Program Change (PC), Control Change (CC) messages to remotely change presets or to control other functions or parameters in Solis Ventus. The MIDI input accepts Type-A TRS-to-DIN5 pin mapping:
