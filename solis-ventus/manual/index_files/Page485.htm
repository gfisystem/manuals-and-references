5.3  Expression & Control Voltage

Two separate expression inputs are available for use:

· EXP1 (1/4” jack).

· EXP2 (1/8” jack).

EXP1 input is shared with Aux Switch input. User must configure the AUX/EXP1 input to be used for either Aux Switches or for Expression pedal, this is done in the Global Settings menu.

 

EXP2 input can also accept Control Voltage (CV) sources, with switchable voltage range (5V/ 10V). When not in use it’s recommended to disable EXP2 input to avoid spurious false sensing, this is done in the Global Settings menu.

 

Expression inputs are optimized for 10K pot resistance value and “Tip-to-Wiper” polarity.
